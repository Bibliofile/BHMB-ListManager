List Editor

+

This extension lets you define and apply lists with commands. This can be used to apply a lockdown, whitelisting well known players, or reset staff lists.

Warning

Adding a user to a list DOES NOT automatically add them to the server list.

Commands

Commands can only be run by the server owner and SERVER.

  • /list apply <list id> - apply a list, will overwrite the existing server list.
  • /list reset <list id> - resets the server list to the previous list. When the bot is reloaded, previous lists are lost.
